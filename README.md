## 🛠️ Instalación local

### Requisitos

- [Node.js](https://nodejs.org/) 18+

### Clonar e iniciar el proyecto

```bash
git clone https://github.com/tu-usuario/nombre-del-repo.git
cd nombre-del-repo
npm install
npm run start
```
Luego visita (http://localhost:3000) para ver la documentación localmente.
